# 📊 Big Mart Sales Analysis

A comprehensive **data analysis project** that explores the Big Mart sales dataset to uncover meaningful business insights and identify the factors influencing product sales across different outlets. Using Python and popular data analysis libraries, this project focuses on data cleaning, exploratory data analysis (EDA), visualization, and interpreting sales patterns to support data-driven decision-making. The commonly used Big Mart dataset contains sales information for products across multiple stores and is widely used for retail analytics and predictive modeling.

This project demonstrates my ability to transform raw data into actionable insights while applying data analysis techniques commonly used in real-world business scenarios.

---

## 📖 Project Overview

Understanding customer purchasing behavior and sales performance is essential for making informed business decisions. This project analyzes the Big Mart sales dataset to identify trends, evaluate product and outlet performance, and explore the relationship between different features that influence sales.

The analysis includes data preprocessing, handling missing values, exploratory data analysis, and visualizations that help uncover patterns within the dataset.

## 🚀 Getting Started

### Prerequisites

Before running the project, ensure you have:

* Python 3.x
* Jupyter Notebook or JupyterLab
* Required Python libraries

### Installation

Clone the repository:

```bash
git clone https://github.com/NancyMerciline/Big-Mart-Sales-Analysis.git
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook and open the project notebook to explore the complete analysis.
